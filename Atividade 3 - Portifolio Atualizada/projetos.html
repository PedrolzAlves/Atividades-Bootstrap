Jogo da forca em C++
Jogo da Forca em C++

Projeto desenvolvido com a linguagem c++.

Basicamente consiste em uma replicação do jogo de forca. O recomendado é jogar com duas pessoas. Um digita uma palavra secreta sem o outro ver. A partir desse momento o que não viu a palavra tenta descobrir o segredo. Apenas 5 erros podem ser cometidos. Digitar uma letra errada ou chutar a palavra errada, dimui uma tentativa. O jogo termina caso o jogador adivinhar a palavra secreta ou as 5 tentativas acabarem.

Foram implementados um sistema para aparecer as letras erradas e um contador na tela com o número de tentativas restantes.

Lixeira Inteligente
Lixeira inteligente

Projeto desenvolvido com a linguagem c++ utilizada na plca arduino uno.

A ideia do projeto era conseguir criar um protótipo de uma lixeira inteligente utilizando uma placa arduino e outros componentes eletrônicos. Os requisitos do projeto era criar uma lixeira de fácil uso para pessoas com deficiências auditivas, daltonismo, e outros problemas visuais, e ainda para pessoas que não sabem ler.

O projeto funciona da seguinte forma: O circuito possui um sensor de movimento que quando alguem chegar perto automaticamente a tampa da lixeira, que é simulada pelo servo motor. Para cegos quando a pessoa está próxima da lixeira um som com o nome do material da lixeira é emitido. Para surdos há cores e que ascendem representando cada material reciclado, além do display LED que ficaria em frente de cada de lixeira com seu nome, servindo para surdos, e daltônicos

Jogo Gênius
Jogo Gênius

Projeto desenvolvido com a linguagem c++ utilizada na placa arduino.

A proposta deste projeto era recriar o conhecido jogo de memória Gênius utilizando componentes eletrônicos controlados por uma placa arduino.

A placa arduino uno possui um código programado em c++ que controla, leds, botões e buzzers para replicar uma sequência. Assim que energizada os leds ascendem e o jogador pode apertar o botão para iniciar o jogo. Assim que iniciado um led aleatório ascenderá, o jogador então deve apertar o botão do led que ascendeu. Caso acerte, a sequência recomeça acrescentando mais um led aleatória, e assim o processo se repete.

O Jogo termina caso o usuário erre a sequência ou consiga acertar um total de dez sequências seguidas.
